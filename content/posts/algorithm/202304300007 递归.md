#算法

递归能够解决的问题的**特点**：
- 原问题能够分解成一个更小的与原问题解决思路相同的子问题

递归的**步骤**：
1. 拆分问题，找使用子问题解决原问题的方法（原问题与子问题的关系）
2. 找子问题的解决方法，当与原问题的解决方法一致，便能够递归
3. 结束的条件
4. 时间复杂度

可以将**编码的过程**总结为：
```
1. 定义方法
2. 写明结束条件
3. 根据原问题、子问题之间的逻辑关系写主要的递归逻辑
```

递归存在的**问题**：
1. 栈溢出
2. 重复计算
